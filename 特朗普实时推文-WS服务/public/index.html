Donald Trump · Truth Social 实时推文 WebSocket 推送

多源轮询 Trump 在 Truth Social 的发帖，秒级通过 WebSocket 推送给已授权客户端。 返回字段包含英文原文与中文翻译，可直接喂给 AI 模型或量化策略。

数据链路

环节典型耗时说明
Truth Social 发帖 → 上游索引 30 – 80 秒 免费源（trumpstruth.org / Mastodon RSS）的固有节奏
上游 → 本服务轮询命中 0 – 5 秒 多源并发，最快者赢；HTTP ETag / If-Modified-Since
本服务 → WebSocket 客户端 < 50 ms 单次 JSON 帧，无队列

合计端到端典型 30 – 90 秒。这是当前免费数据源的物理下界 — 想压到 sub-10 秒需付费上游或绑 Truth Social 账号。

使用流程

  1. 到 /admin 登录后台，生成一把 API Key（明文仅显示一次，请立即保存）
  2. 客户端用 ws://...?token=... 建立 WebSocket 连接
  3. 连接建立后服务端先发 hello 帧；客户端可发 {"type":"history","limit":50} 拉取历史
  4. 有新帖时服务端推送 {"type":"tweet","data":{...}}，字段见 文档
  5. Key 撤销 / 过期时所有该 key 的连接会被服务端踢断（close code 4003）
